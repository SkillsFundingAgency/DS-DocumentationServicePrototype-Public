  1. Contents

Up-to-date guidance for submitting Individualised Learner Reports (ILR)

  1. Recording traineeship programmes Introduction to Individualised Learner Reports (ILR)

    Find out about ILR, what ILR data used for and who is required to send us ILR data returns.
  2. Collecting and submitting ILR data

    Five data management principles must be adhered to when submitting and ILR. Find out how to collect and when to send ILR data.
  3. Guidance for all provision

    What to include and what not to include when completing the final return for the current year.
  4. Recording learner changes

    Guidance about:
    • Correcting errors.
    • Recording changes to a learner’s details.
    • Recording changes to a learner’s programme or learning aim.
  5. Recording apprenticeship programmes

    Guidance for all apprenticeship programmes and examples.
  6. Recording traineeship programmes

    Guidance for recording Traineeships for 16 to 24 year olds including:
    • Work experience , work preparation.
    • Completion, achievement and retention.
    • Early leavers and transfers.
    • Destination and progression information.
    Includes examples on how to record the Outcome fields in different scenarios.
  7. Recording 16-19 study programmes

    16-19 funded study programmes are recorded on the ILR using Funding model code 25. A programme aim record (Aim type = 1) is not required for these study programme unless the learner is doing a traineeship programme.
  8. ESF funded learning aims

    For the ESF 2014 to 2020 Programme, all records for all learners who have received ESF funding as part of the current programme must continue to be returned for the duration of the ESF programme.
  9. Recording A and AS levels

    You must accurately record the learning aim that a learner is working towards: either an AS or A level learning aim.
  10. Recording Household situation

    Used to record the household situation that applies at the start of each learning aim. This does not need to be updated if the household situation subsequently changes. Find out when to use household situation codes.
  11. Field collection requirements tables

    The tables in this section list the fields that are collected for the different funding models. Not all fields listed will need to be completed for all learners in all circumstances.
