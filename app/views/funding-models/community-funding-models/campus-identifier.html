Campus identifier: field

Part of the learner entity

Overview

You should return only one record for each learner. The data recorded in the learner entity contains basic information specific to the learner such as their name, date of birth, sex and ethnicity.

Version 2 changes: Updates Published March 2018   VERSION 2

What's changed?

Pattern has been corrected to [A-Za-z0-9]{1,8}

The pattern was incorrect in version 1.
“8” is the change to the pattern in version 2.

Definition

The identifier of a campus within a college group and no longer a separate legal entity but previously operated as an incorporated college with a UKPRN.

Reason required

To allow identification of provision delivered across the various sites of merged institutions.

Schema definitions

XML Element Name CampId
Field length 8
Data type RestrictedString
Minimum occurrences 0
Maximum occurrences 1
Part of Learner

Valid entries

Must be in a valid look up. This is a 8-digit identifier using alphanumeric characters.
Pattern    VERSION 2
[A-Za-z0-9]{1,8}

Collection requirements

This field is collected for all learners.

Notes and guidance about this field

This field records the identifier of a campus within a college group who is no longer a separate legal entity but previously operated as an incorporated college with a UKPRN

This should only be returned where the provider has been notified that they can use this field.

The campus identifier must be linked to a college group with a separate UKPRN.
